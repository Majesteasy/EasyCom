## RÈGLES DE TOURNAGE (pour InVideo AI / Pictory)

1. **Sous-titres obligatoires** — 80% regardent sans son
2. **Texte à l'écran** duplique ce qui est dit (hook visible en grand)
3. **Musique** : tendance TikTok + beat drop sur la révélation produit
4. **Plan produit** : gros plan 2 secondes minimum sur le produit physique
5. **Lien dans description YouTube** : `amazon.fr/dp/{ASIN}?tag=easycom0ae-21`
6. **Mention légale stories** : #ad ou #partenariat si rémunération directe (affilié = optionnel en Suisse mais recommandé)
